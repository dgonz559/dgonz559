What is code?

Code is a system of words, numbers, letters, figures, or symbols substituted for other words, numbers, letters, or symbols. In the case of Web Development we use different languages to program instructions, adjust the web content we share to our liking and add functionality for our users. Although this process may seem complex, the work is broken down into different sections. Depending on what part of your project you're focused on, you can figure out which language to use and manipulate web content accordingly.

There are different languages used for specific tasks in the development process. Front end developers work with languages that render in the browser, also called a client. HTML, CSS , and Javascrpt are commonly used front end languages. Back end development customizes content for each user depending on data and might require a special server. Servers manage data and how it's presented to the user. A popular combination of back end languages is PHP and SQL.
